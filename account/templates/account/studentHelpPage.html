QUIZIT

QUIZIT is a educational website created which primarily works as a question answer site. Students can enroll into available courses and can take quizzes set by the instructor for the enrolled courses.

QUIZIT is one of the CS educational technologies created by CSI Lab of School of Computing, Informatics & Decision Systems Engineering at Arizona State University. Some features of the website are listed below:

User Authentication

QUIZIT allows a student to register in the website for a course. The system authenticates a student using their email id and password. Students can also use the Forgot Password feature to reset the password

 

Today's Quiz

QUIZIT's student portal allows a student to view the quiz of the day when logged in. The student can select an option and submit that as an answer.

 

Calendar View

QUIZIT's student portal displays a calendar view of the quizzes available in the today's quiz page. The coloring scheme for the calendar is described in the legends adjacent to the calendar. Clicking on a date will display the question and let the user retake or review a quiz.

 

Question History

QUIZIT's student portal displays a calendar view of the quizzes available in the today's quiz page. The coloring scheme for the calendar is described in the legends adjacent to the calendar. Clicking on a date will display the question and let the user retake or review a quiz.

 

Review/Retry

QUIZIT's student portal lets a student review a question where they can view previously attempted choices. A student can also retry a quiz which will let them practise the question over and over again. Using the retry option is counted as an attempt. The Review/Retry pages also has a section where students can drop comments on the question and view comments by other students.
